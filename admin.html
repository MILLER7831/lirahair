Для автоматической публикации на Netlify, подключите GitHub репозиторий.

Инструкция по настройке:

  1. Создайте репозиторий на GitHub
  2. В Netlify подключите этот репозиторий
  3. Создайте Personal Access Token в GitHub Settings
  4. Вставьте токен ниже
Токен сохраняется только в вашем браузере
